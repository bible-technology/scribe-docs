The left-side column can load different reference resources inside sections and it can have up to two sections. 

Available reference resources in Autographa

- **Bible**
- **Translation Notes**
- **Translation Words**
- **Translation Questions**
- **Translation Academy**
- **Open Bible Stories**
- **OBS Translation Notes**
- **OBS Translation Questions**
