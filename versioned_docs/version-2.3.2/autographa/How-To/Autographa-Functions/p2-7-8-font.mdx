title: How to Change Font Size
---

To change the Font Size of your project first, 

- Find the **text** icon <img src="/autographaAssetsV2/how-to/font-soze.png" width="50px" alt=""/> on the top right corner of the **Translation column.**
- Select the small letter A to make the font size smaller.
- Select the big letter A to make the font size larger.
