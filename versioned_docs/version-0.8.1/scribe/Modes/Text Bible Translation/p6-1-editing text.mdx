The Text Editor allows users to type, edit, or translate text for Bible translation projects. 

### Getting Started 

 - To begin a new project, click the **New** button from the left side menu on the project page. 
 - On the New Project page, open the drop-down menu to select the project flavor. 
 - Choose **Bible Translation** from the list and proceed to create the project. 
 - Fill in all the required information in the fields. 
 - Click on **Advanced Settings** to select books from four available options: 
   - All Books 
   - Old Testament 
   - New Testament 
   - Custom 
 - To customize your selection, choose Custom Books and select the books as per your preference. 
 - Click the **Save** button. 
 - Click on **Create Project.**A pop-up message will confirm that the project has been created successfully. 

<video controls src="/0.8.1/en_texteditor_create.mp4" width="100%" type="video/mov"/>

### Working in the Text Editor 

 - Open the **Project** you created; it will launch with the **Text Editor**. 
 - Click on the icon for **new layout** to load reference resources.
 - Click the **resource selector** icon on the **new layout.**
 - A **Resource pane** will open up, displaying a list of available resources.
 - Use the **BCV Navigator** to switch between **Book, Chapter, and Verse**. 
 - Select the verse you wish to **type, edit, or copy-paste** into. 

<video controls src="/0.8.1/en_texteditor_working.mp4" width="100%" type="video/mov"/>

### Shortcut Keys for Text editor 

- **Ctrl+Z** – Undo 
- **Ctrl+y** – Redo 
- **Tab** - Next verse 
- **Shift+tab** – Previous Verse 
- **Ctrl+s** - Save 

### Font Adjustments in Text Editor 

To change the font style, click the <img src="/0.8.1/en_font-style.png"  width="30px" alt="fontstyle"/> icon at the top-right corner of the editor. 
To adjust the font size, click the <img src="/0.8.1/en_font-size.png"  width="30px" alt="fontsize"/> icon at the top-right corner.
