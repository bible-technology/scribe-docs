- ಹೊಸ ಲೇಔಟ್ ಐಕಾನ್ ಮುಂದೆ ಪರದೆಯ ಮೇಲಿನ ಬಲಭಾಗದಲ್ಲಿರುವ **ಬೆಲ್** ಐಕಾನ್ ಮೇಲೆ ಕ್ಲಿಕ್ ಮಾಡಿ<img src="/assets/notification.png"  width="50px" alt="notification"/>
